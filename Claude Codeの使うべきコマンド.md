### 1. 最重要コマンド（これだけは覚えるべき）

これらは日常的に使用頻度が高く、開発効率に直結します。

- **`/init`**
    - **機能**: プロジェクトの初期設定を行い、`CLAUDE.md`（プロジェクトの説明書）を自動生成します。
    - **用途**: 新しいプロジェクトで最初に実行し、AIにプロジェクトの概要やルールを理解させるために使います。
- **`/compact`**
    - **機能**: これまでの会話履歴を要約してコンテキスト（メモリ）を圧縮します。
    - **用途**: 会話が長くなり、AIの動作が重くなったり、トークン消費（コスト）を抑えたい時に使います。「文脈が消える」のを防ぎつつ、容量を節約できます。
- **`/clear`**
    - **機能**: 会話履歴を消去します。
    - **用途**: 話題を完全に切り替えたい時や、コンテキストが溢れそうな時に使います。ただし、これまでの文脈は完全に失われます。
- **`Shift + Tab` (ショートカット)**
    - **機能**: **「Plan Mode（計画モード）」**への切り替え。
    - **用途**: いきなりコードを書かせるのではなく、「どういう手順で実装するか」をAIに計画させたい時に使います。品質担保のために非常に重要視されています。

### 2. 作業の手戻りを防ぐ・修正するコマンド

AIが間違った方向に行った場合や、状態を戻したい時に使います。

- **`/rewind`** （または `Esc` 2回押し）
    - **機能**: 会話やコードの状態を過去の特定の時点まで巻き戻します。
    - **用途**: 「指示を間違えた」「バグが埋め込まれた」という場合に、その前の状態に戻してやり直すことができます。Gitのリセットに近い感覚です。
- **`/model`**
    - **機能**: 使用するAIモデル（Claude 3.5 SonnetやOpusなど）を切り替えます。
    - **用途**: 普段は高速なSonnetを使い、複雑な設計や計画が必要な時だけ賢い「Opus」に切り替えるといった使い分けが可能です。

### 3. 情報確認・設定コマンド

現状の把握やツールの設定変更に使います。

- **`/stats`**
    - **機能**: 直近の利用状況やトークン消費量を確認します。
    - **用途**: どれくらいコストがかかっているか、制限に近づいていないかを視覚的に確認できます。
- **`/config`**
    - **機能**: 各種設定を確認・変更します。
    - **用途**: 例えば「Thinking Mode（思考プロセスを表示する機能）」のオン/オフ切り替えなどが可能です。
- **`/mcp`**
    - **機能**: MCP（外部ツール連携）サーバーの状態確認や有効化/無効化を行います。
    - **用途**: ブラウザ操作（Playwright）やDB操作など、特定のツールを使わせたい時に確認します。

### 4. 起動時のオプション（ターミナルで入力）

Claude Codeを起動する際に入力するコマンドです。

- **`claude --resume`** (または `/resume`)
    - **機能**: 過去のセッション（会話履歴）を引き継いで起動します。
    - **用途**: 前回の作業の続きから始めたい時に使います。
- **`--dangerous-skip-permissions`**
    - **機能**: ファイル編集やコマンド実行のたびに求められる「許可（Y/N）」をスキップします。
    - **用途**: **上級者向け**。いちいち承認するのが面倒な場合に、全自動で作業させるために使います（ただし、勝手にファイルを消されるリスクもあるため注意が必要です）。

### 5. その他便利なテクニック

- **カスタムコマンド（例: `/commit`）**
    - 自分で好きなコマンドを作成できます。例えば `/commit` と打つだけで、「変更内容を解析してコミットメッセージを作成し、Gitコミットする」といった一連の動作を登録可能です。
- **マジックワード: `UltraSync`**
    - コマンドではありませんが、プロンプトにこれを含めると、Claudeが「徹底的に考えて」回答を生成するようになります（Thinking Tokenを多く使う挙動になるようです）。

まずは **`/init`** で始め、**`Shift+Tab`** で計画を立て、長くなったら **`/compact`** する、という流れを覚えるのが基本となります。
